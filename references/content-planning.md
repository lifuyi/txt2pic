# 内容规划方法论

## 核心原则

**内容驱动，形式跟随内容**。不预设固定模板，根据用户提供的具体内容灵活规划图文卡片。

---

## 内容分析流程

### 1. 理解内容本质

**分析维度**：

| 维度 | 问题 | 影响 |
|------|------|------|
| 主题 | 内容在讲什么？ | 决定图片选择 |
| 基调 | 情感色彩如何？ | 决定视觉风格 |
| 密度 | 信息量多大？ | 决定图片数量和每张信息量 |
| 结构 | 如何组织？ | 决定内容分配方式 |
| 受众 | 给谁看？ | 决定文案风格 |

### 2. 提取核心信息点

**方法**：
1. 通读内容，圈出所有信息点
2. 按重要性排序
3. 合并相似点，拆解复杂点
4. 确定每张图承载 1 个核心信息点

**示例**：

用户提供内容：
> 浙江大学宿舍是4人间，上床下桌，有空调、独立卫浴，每层有公共厨房和洗衣房。住宿费1200-1600元/年。宿舍区附近有便利店、快递柜。

提取信息点：
1. 房型配置（4人间，上床下桌）
2. 设施配备（空调、独立卫浴）
3. 公共区域（厨房、洗衣房）
4. 费用（1200-1600元/年）
5. 生活便利（便利店、快递柜）

规划图片：
- P1 封面：宿舍概览 + 核心卖点
- P2：房型展示
- P3：设施配备
- P4：公共区域
- P5：生活便利 + 费用总结

## 信息密度与图片数量

### 判断信息密度

| 密度 | 特征 | 每张字数 | 图片数 |
|------|------|----------|--------|
| 轻量 | 单一主题，要点清晰 | 60-100 | 5-7 |
| 中等 | 多维度展开 | 100-180 | 7-12 |
| 重量 | 信息密集，数据多 | 150-250 | 10-18 |

### 信息量平衡原则

**信息多 → 视觉简洁**
- 文字内容丰富时，背景和装饰要简洁
- 使用留白、清晰分区帮助阅读

**视觉强 → 文字精炼**
- 图片展示性强时，文字要精简
- 突出图片氛围，文字点缀

## 内容与视觉匹配

### 根据内容选择风格

```
内容特征分析 → 推荐风格

年轻/活力/生活化 → 青春活力风
数据/排名/官方 → 专业权威风
简洁/现代/设计感 → 简约现代风
历史/文化/底蕴 → 学院复古风
自然/环境/生态 → 清新自然风
```

### 根据内容选择布局

```
内容特征分析 → 推荐布局

需要展示特定图片 → 分割布局（图上文下或图左文右）
信息量大需层次 → 卡片镂空（图片背景 + 内容卡片）
氛围感强的图片 → 渐变遮罩（图片全背景 + 文字叠加）
追求设计简洁 → 边框留白（内容居中 + 图片边框装饰）
```

### 内容与图片匹配

**原则**：图片要服务于内容，不是随便放

| 内容主题 | 推荐图片 | 避免 |
|----------|----------|------|
| 宿舍条件 | 宿舍实景、公共区域 | 校门、图书馆 |
| 食堂餐饮 | 食堂环境、菜品 | 运动场 |
| 专业介绍 | 实验室、课堂 | 宿舍生活 |
| 校园环境 | 风景、建筑 | 室内设施 |

## 图片序列规划

### 通用框架（可调整）

**封面图**：
- 主题标题
- 核心亮点（一句话）
- 主视觉图片

**内容图**：
- 每张承载 1 个核心信息点
- 信息点之间有逻辑关联
- 视觉节奏有变化（信息密度交替）

**收尾图**（可选）：
- 总结 / 金句
- 互动引导

### 节奏控制

避免所有图片信息密度相同，建议交替：

```
封面：高视觉冲击
内容1：高信息密度
内容2：中信息密度（稍轻松）
内容3：高信息密度
内容4：低信息密度（视觉为主）
收尾：情感共鸣
```

## 文案改写技巧

### 小红书化表达

**口语化**：
- ❌ 「该校宿舍配置较为完善」
- ✅ 「宿舍配置超齐全！」

**情绪词**：
- 「超」「真的」「绝了」「太香了」

**互动感**：
- 「评论区告诉我你的学校宿舍怎么样👇」
- 「收藏起来填志愿用❤️」

**符号使用**：
- 标签：✨ 📚 🏠 💡 🎯
- 分隔：| / →
- 强调：【】 ❶❷❸

### 标题公式

- 数字 + 痛点：「6个宿舍细节，选校必看」
- 对比 + 结果：「从不知道到全了解，看这篇就够了」
- 悬念 + 答案：「这所大学的宿舍凭什么这么香？」
