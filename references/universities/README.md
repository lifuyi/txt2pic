# 大学图片库

存放各大学的图片素材，供图文生成时引用。

## 文件夹结构

```
universities/
├── 浙江大学/
│   ├── logo.png           # 学校 logo
│   ├── campus-1.jpg       # 校园图片
│   ├── campus-2.jpg
│   └── ...
├── 南京大学/
│   ├── logo.png
│   ├── campus-1.jpg
│   └── ...
└── ...
```

## 命名规范

### 文件夹命名

使用大学全称：
- ✅ `浙江大学`
- ✅ `南京大学`
- ❌ `浙大`、`NJU`

### 文件命名

| 文件名 | 说明 |
|--------|------|
| `logo.png` | 学校官方 logo（必须，用于品牌标识） |
| `campus-*.jpg` | 校园图片，数字编号即可 |

**图片内容建议涵盖**：
- 校门/标志性建筑
- 教学楼/图书馆
- 宿舍/食堂
- 运动场/体育馆
- 校园风景

但不强制命名，用户可自由命名，使用时根据内容选择匹配的图片。

## 图片要求

- **格式**：JPG 或 PNG
- **logo**：建议 PNG（透明背景）
- **校园图片**：建议 JPG，横版竖版均可
- **分辨率**：建议 1920px 以上

## 使用方式

生成图文时，根据内容主题选择匹配的图片：

**示例**：用户要生成「浙江大学宿舍条件」图文
1. 读取 `浙江大学/` 文件夹
2. 选择宿舍相关的图片（如 `campus-3.jpg`）
3. 如果需要 logo，引用 `logo.png`

**Prompt 中引用**：
```
**University Image**: references/universities/浙江大学/campus-3.jpg
(宿舍楼外观，阳光下的建筑)
```

## 图片来源建议

1. 大学官网 / 官方公众号
2. 摄影网站（Unsplash、Pexels，注意版权）
3. 实地拍摄（原创，版权无忧）
